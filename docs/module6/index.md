# Module 6 — Wrap-Up Challenge

## Overview

Review and reinforce all skills covered in this course
with 50 practice problems across two sets. Work through
each problem independently, check your solutions, and
review any missed questions before your next attempt.

## How the Sets Differ

**Practice Set 1** is organized by topic with hints and
walkthroughs. Use it to identify which areas need review
and to build confidence before the challenge set.

**Practice Set 2** is randomized with harder numbers and
no hints — designed to simulate a clinical competency exam.
Aim for 90% or higher (23 out of 25).
