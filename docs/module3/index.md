# Module 3 - Ratio and Proportion

## Overview

Ratio and proportion form the mathematical backbone of nearly 
every dosage calculation in nursing. Once you can confidently 
set up and solve a proportion, most medication calculations 
become straightforward.

This module builds directly on the conversion and dimensional 
analysis skills from Module 2.

## Learning Objectives

By the end of this module you should be able to:

- Define and write ratios in multiple formats
- Set up and solve proportions
- Use cross-multiplication to find unknown values
- Apply ratio and proportion to clinical dosage problems
- Recognize when an answer is unreasonable
