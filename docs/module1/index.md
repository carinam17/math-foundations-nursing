# M1 · Arithmetic

## Overview

This module covers the number operations that appear inside every
clinical calculation. You will learn how to work with whole numbers, fractions, decimals, rounding, and percentages.

Work through each topic in order, or use the self-check at the end
of each page to confirm you're ready and move on.

## Learning Objectives

By the end of this module you will be able to:

- Perform operations with whole numbers, fractions, and decimals
- Round numbers to the required precision using nursing standards
- Read and calculate with percentages and solution concentrations
- Recognize fractions as relationships between quantities. 


## Topics

| Topic | Key Skills |
|---|---|
| [Whole Numbers](whole-numbers.md) | Place value, multiplication and division, powers of ten, estimation |
| [Fractions](fractions.md) | Reading a fraction as a factor, multiplying, simplifying |
| [Decimals](decimals.md) | Place value, safe notation, connecting decimals to metric units |
| [Rounding](rounding.md) | Nearest whole, tenth, hundredth rounding |
| [Percentages](percentages.md) | Converting, calculating, reading solution concentrations |


## Estimated Time

Approximately 1–2 hours including practice problems.

!!! tip "Before You Begin"
    Have a pen and paper ready. Attempt every practice problem
    by hand before checking the answer. Writing out each step
    builds the habit of showing and verifying your work.
